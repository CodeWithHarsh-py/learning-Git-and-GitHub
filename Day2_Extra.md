To delete a branch 👇

1. 🗑️ Local branch delete [ Deletes branch safely (only if merged). ]

*code =>* git branch -d branch_name    

2. If you want to force delete (even if not merged):

*code =>* git branch -D branch_name

3. 🌐 Remote branch delete:

*code =>* git push origin --delete branch_name
